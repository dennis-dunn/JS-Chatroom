How to use the JS-Chatroom:
Author: Dennis Dunn
May 18, 2018
1) Make sure that node.js is installed: https://nodejs.org/en/
2) In a command console, run "node mainServer.js" to start the backend server.
3) Open client.html in the browser (2 or more instances on any machine)
4) The server will run on the default port 8080, but enter the IP address with this port number
	then choose a UserName and click "Connect" for all the clients.
5) Type a message into the textbox and press "Send" to distribute it among all clients
